MINHAS TECNOLOGIAS.

HTML

HTML (Linguagem de Marcação de Hipertexto) é a linguagem utilizada para criar e estruturar páginas da web, fornecendo a base para o conteúdo e permitindo a organização de elementos como texto, imagens e links por meio de tags.

CSS

CSS (Folha de Estilo em Cascata) é uma linguagem de estilo usada para definir a apresentação e o design de páginas da web, separando o conteúdo estrutural do seu estilo visual.

JAVASCRIPT

JavaScript é uma linguagem de programação usada para tornar páginas da web interativas e dinâmicas, permitindo manipulação de conteúdo, interação com usuários e criação de experiências envolventes na web.
